Digital Distillery Private Key Repair

Private Key Repair will repair a faulty private key (Compressed/uncompressed Base58 or Hexadecimal).

  • You need to supply the damaged key in a list of one or more possible damaged keys
  • If there is more than one key in the list, add --max-tokens 1
  • Substitute missing/unknown hexadecimal characters with %H
  • Substitute missing/unknown base58 characters with %B
  • Indicate missing character (%H), two-character (%2H) or three-character (%3H) etc.
  • If there are only typos (no missing characters) you will need a tokenlist
  • If there are typos and missing characters you will need a passwordlist
  • You will also need a known address or an address database file
